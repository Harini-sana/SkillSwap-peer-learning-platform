# SkillSwap – Peer Learning Platform

SkillSwap is a peer-to-peer skill exchange platform where users can **learn new skills by teaching others**.
Instead of paying for courses, users can connect with people who have different expertise and exchange knowledge through collaborative learning.

The platform encourages **community learning, skill sharing, and personal growth**.


## 🚀 Key Features

* 👤 **User Profiles**

  * Users can create profiles with skills, bio, and learning interests.

* 🔍 **Explore Skills Feed**

  * Users can view posts from other learners and discover new skills.

* 🤝 **Skill Exchange Requests**

  * Send and receive skill exchange requests between users.

* 💬 **Real-Time Messaging**

  * Chat with exchange partners for collaboration and learning.

* 📊 **Learning Progress Tracking**

  * Track roadmaps, completed steps, and learning streaks.

* 📅 **Calendar & Weekly Reflection**

  * Monitor learning activity and reflections.

* 🏆 **Certificates**

  * Users receive certificates after completing skill exchanges.

* ⭐ **User Ratings**

  * Rate learning partners after exchanges.

* 📈 **Profile Insights**

  * Shows profile views, posts, likes, and activity stats.

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository


git clone https://github.com/Harini-sana/SkillSwap-peer-learning-platform.git


### 2️⃣ Navigate into the project


cd SkillSwap-peer-learning-platform


### 3️⃣ Create virtual environment


python -m venv venv


### 4️⃣ Activate virtual environment

Windows:


venv\Scripts\activate


### 5️⃣ Install dependencies


pip install django


### 6️⃣ Run migrations


python manage.py migrate


### 7️⃣ Start the server


python manage.py runserver


Then open:


http://127.0.0.1:8000
